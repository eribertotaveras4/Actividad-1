1. Que es un Sistema Operativo?

Es la parte intangible o la parte logica de un computador.

2. Componentes de un Sistema Operativo?

Los principales componentes son el Kernel y los nucleos cada unos de ellos incluyen una parte fundamental en su funcion de sistema operativo y se ejecuta en modo privilegiado

3. Tareas del Sistema Operativo?

La tarea principal es administrar los programas

4.Funciones de un Sistema Operativo ?

Su funcion es gestionar los recursos del sistema permitiendo ejecucion de aplicaciones, programas y demas

5. Caracteristicas de un Sistema Operativo ?

-Intermediario entre el hardware y el usuario.

-Seguridad

-Administrar

-Interacion, ETC.

6. Tipos de Sistema Operativo ?

-Multiusuario; Permite que varios usuarios puedan ejecutar simultaneamente

-Monousuario; Permite ejecutar solo un usuario a la vez

-Monotarea; Permite ejecutar un solo proceso a la vez

Multirarea, Puede ejecutar varios procesos al mismo tiempo

9. Sistema Operativo segun el Ordenador y el Tipo de Aplicaciones?

-Administra memoria

14. Que es un sistema operativo multitareas ?

Es un sistema operativo que puede ejecutar varios procesos a la vez

15. Que es un sistema operativo multiusuario ?

Permite que varios usuarios puedan ejecutar simultaneamente

16. Como funciona un sistema operativo?

Funciona como intermediario entre el hardware y el software de aplicacion

17. Como se utiliza un sistema operativo en un entorno grafico?

Un sistema operativo de ambiente grafico se basa en imagenes e iconos. Se caracteriza por tener un caracter mas intuitivo para el usuario a traves del uso de lenguaje escrito e imagenes

18. Cuando inicia el funcionamiento del Sistema Operativo ?

Cuando se enciende la PC

19. Mencione las dos interfaces que forman parte del Sistema Operativo ?

-Interfaz de Usuario(UI)

-Interfaz de programacion de aplicaciones (API)

20. Que es un proceso en entorno de los sistemas operativos?

Un proceso en el entorno de los sistemas operativos es una instancia en ejecucion de un programa, con su propio espacio de memoria y recursos asignados

21. Que es un sistema operativo distribuido y centralizado?

Un sistema operativo centralizado gestiona todos los recursos desde un solo punto, mientras que un sistema operativo distribuido coordina recursos y tareas entre multiples nodos de una red.

22. Mencione los sistemas de archivos utilizados por los Sistemas Operativos ?

NTFS

FAT32

23. Mencione otras de las caracteristicas actuales de los sistemas operativos?

Otras caracteristicas de los sistemas operativos modernos incluyen la virtualizacion, multitarea, seguridad avanzada, administracion de energia, gestion de redes, recuperacion ante fallos, actualizaciones automaticas y una interfaz de usuario intuitiva.

24. Que permite la exigencia de muchos controladores en un sistema operativo.?

Permite que el sistema pueda comunicarse y gestionar una amplia variedad de dispositivos de hardware, como impresoras, tarjetas de red, tarjetas graficas, etc.
