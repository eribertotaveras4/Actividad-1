1. ¿Qué es un Sistema Operativo?

Es la parte intangible o la parte logica de un computador.

2. ¿Componentes de un Sistema Operativo?

Los principales componentes son el Kernel y los nucleos cada unos de ellos incluyen una parte fundamental en su funcion de sistema operativo y se ejecuta en modo privilegiado

3. ¿Tareas del Sistema Operativo?

La tarea principal es administrar los programas

4.¿Funciones de un Sistema Operativo ?

Su funcion es gestionar los recursos del sistema permitiendo ejecucion de aplicaciones, programas y demas

5. ¿Características de un Sistema Operativo ?

-Intermediario entre el hardware y el usuario.

-Seguridad

-Administrar

-Interacion, ETC.

6. ¿Tipos de Sistema Operativo ?

-Multiusuario; Permite que varios usuarios puedan ejecutar simultaneamente

-Monousuario; Permite ejecutar solo un usuario a la vez

-Monotarea; Permite ejecutar un solo proceso a la vez

Multirarea, Puede ejecutar varios procesos al mismo tiempo

9. ¿Sistema Operativo según el Ordenador y el Tipo de Aplicaciones?

Conjunto de programas que permite manejar la memoria, disco, medios de almacenamiento.

10. ¿Compatibilidad de los Sistema Operativo?

Windows; Es el sistema operativo mas compatible con los dispositivos

11. ¿Cuál es la función básica de un Sistema Operativo ?

Luego de ser la parte intangible de un ordenador, esta encargada de administrar los diferenntes programas

-Windows

13. ¿Menciona algunas tareas que realizan los Sistemas Operativos ?

-Almacenamiento

-Administra memoria

-Gestion de archivos

-Manejos de errores

-Interfaz de usuarios

14. ¿Qué es un sistema operativo multitareas ?

Es un sistema operativo que puede ejecutar varios procesos a la vez

15. ¿Qué es un sistema operativo multiusuario ?

Permite que varios usuarios puedan ejecutar simultaneamente

16. ¿Cómo funciona un sistema operativo?

Funciona como intermediario entre el hardware y el software de aplicacion

17. ¿Cómo se utiliza un sistema operativo en un entorno gráfico?

Un sistema operativo de ambiente gráfico se basa en imágenes e iconos. Se caracteriza por tener un carácter más intuitivo para el usuario a través del uso de lenguaje escrito e imágenes

18. ¿Cuándo inicia el funcionamiento del Sistema Operativo ?

Cuando se enciende la PC

19. ¿Mencione las dos interfaces que forman parte del Sistema Operativo ?

-Interfaz de Usuario(UI)

-Interfaz de programacion de aplicaciones (API)

20. ¿Qué es un proceso en entorno de los sistemas operativos?

Un proceso en el entorno de los sistemas operativos es una instancia en ejecución de un programa, con su propio espacio de memoria y recursos asignados

21. ¿Qué es un sistema operativo distribuido y centralizado?

Un sistema operativo centralizado gestiona todos los recursos desde un solo punto, mientras que un sistema operativo distribuido coordina recursos y tareas entre múltiples nodos de una red.

22. ¿Mencione los sistemas de archivos utilizados por los Sistemas Operativos ?

NTFS

FAT32

23. ¿Mencione otras de las características actuales de los sistemas operativos?

Otras características de los sistemas operativos modernos incluyen la virtualización, multitarea, seguridad avanzada, administración de energía, gestión de redes, recuperación ante fallos, actualizaciones automáticas y una interfaz de usuario intuitiva.

24. ¿Que permite la exigencia de muchos controladores en un sistema operativo.?

Permite que el sistema pueda comunicarse y gestionar una amplia variedad de dispositivos de hardware, como impresoras, tarjetas de red, tarjetas gráficas, etc.
